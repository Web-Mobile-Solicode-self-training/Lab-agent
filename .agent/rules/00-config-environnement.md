# Configuration & Environment
- **Respect .gitignore**: Always check the `.gitignore` file.
- **Access Restriction**: Never read or modify files listed in `.gitignore` (vendor, node_modules, .env, etc.).
- **Project Organization**: All new projects must be created within a subfolder under the root `project/` directory (e.g., `project/my-new-project/index.html`).
