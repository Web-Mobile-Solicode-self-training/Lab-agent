# Workflow: Static Component Development
0. **Initialize**: Create a new directory under `project/` for the feature/project.
1. **Structure**: Define the HTML layout using semantic tags within the new directory.
2. **Design**: Apply Tailwind classes and Preline components.
3. **Polish**: Add custom CSS and micro-animations.
4. **Validation**: Check responsiveness and accessibility.
